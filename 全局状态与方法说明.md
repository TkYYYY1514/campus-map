## 1. Pinia Store

所有 Store 位于 `src/stores/` 目录，通过 `defineStore` 定义，使用前需 import。

### 1.1 poiStore（POI 数据管理）

**文件路径：** `src/stores/poiStore.js`

**引入方式：**
```javascript
import { usePoiStore } from '@/stores/poiStore';
const poiStore = usePoiStore();
```

**状态（State）：**

| 变量 | 类型 | 说明 |
|------|------|------|
| `pois` | `ref([])` | 所有 POI 列表 |
| `selectedPoi` | `ref(null)` | 当前选中的 POI 对象 |
| `isLoading` | `ref(false)` | 加载状态 |
| `error` | `ref(null)` | 错误信息 |

**方法（Actions）：**

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `fetchAllPois()` | 无 | `Promise` | 获取所有 POI 列表 |
| `searchPois(keyword)` | `keyword: string` | `Promise` | 按名称模糊搜索 |
| `createPoi(data)` | `data: Object` | `Promise` | 新增 POI |
| `updatePoi(id, updates)` | `id, updates: Object` | `Promise` | 更新 POI 信息 |
| `deletePoi(id)` | `id: number` | `Promise` | 删除 POI |
| `toggleVisibility(id, isVisible)` | `id, isVisible: boolean` | `Promise` | 切换 POI 显示/隐藏 |
| `selectPoi(poi)` | `poi: Object` | `void` | 选中指定 POI |
| `clearSelection()` | 无 | `void` | 取消选中 |

### 1.2 mapStore（地图全局方法）

**文件路径：** `src/stores/mapStore.js`

**引入方式：**
```javascript
import { useMapStore } from '@/stores/mapStore';
const mapStore = useMapStore();
```

**状态（State）：**

| 变量 | 类型 | 说明 |
|------|------|------|
| `centerOnPoi` | `ref(null)` | 地图 POI 定位函数（由 MapBackground 在初始化时注册） |

**方法（Actions）：**

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `registerCenterOnPoi(fn)` | `fn: Function` | `void` | 注册 centerOnPoi 函数（仅由 MapBackground 内部调用） |

**使用示例：**
```javascript
import { useMapStore } from '@/stores/mapStore';
const mapStore = useMapStore();

// 让地图以平滑动画定位到指定 POI
mapStore.centerOnPoi(poi);
```

> **注意：** `centerOnPoi` 在 `MapBackground.vue` 组件挂载后才会注册到 store 中。如果调用时仍为 `null`，说明地图组件尚未加载完毕。

## 2. Composable（组合式函数）

所有 Composable 位于 `src/components/map/composables/` 目录。

### 2.1 useMapTransform（地图变换逻辑）

**文件路径：** `src/components/map/composables/useMapTransform.js`

**引入方式：**
```javascript
import { useMapTransform } from '@/components/map/composables/useMapTransform';
```

**参数（Options）：**
```javascript
useMapTransform({
  IMAGE_WIDTH: 3000,      // 地图原图宽度
  IMAGE_HEIGHT: 1600,     // 地图原图高度
  mapContainerRef         // 地图容器 DOM 引用（ref）
});
```

**返回的状态（State）：**

| 变量 | 类型 | 说明 |
|------|------|------|
| `scale` | `Ref<number>` | 当前缩放比例（ref） |
| `offsetX` | `Ref<number>` | 地图 X 轴偏移量（ref） |
| `offsetY` | `Ref<number>` | 地图 Y 轴偏移量（ref） |
| `isDragging` | `Ref<boolean>` | 是否正在拖拽（ref） |
| `isTransitioning` | `Ref<boolean>` | 是否正在平滑过渡动画中（ref） |
| `wrapperStyle` | `ComputedRef<Object>` | 地图 wrapper 的 CSS 样式 |

**返回的方法（Functions）：**

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `startDrag(e)` | `MouseEvent` | `void` | 开始拖拽地图 |
| `continueDrag(e)` | `MouseEvent` | `void` | 继续拖拽（需配合 mousemove） |
| `endDrag()` | 无 | `void` | 结束拖拽 |
| `onWheel(e)` | `WheelEvent` | `void` | 鼠标滚轮缩放 |
| `getInitialScale()` | 无 | `number` | 获取初始缩放比例 |
| `getCenterOffset(scale)` | `scale: number` | `{ x, y }` | 获取地图居中时的偏移量 |
| `clampOffset(x, y, scale)` | `x, y, scale` | `{ x, y }` | 限制偏移量在可视区域内 |

### 2.2 useMapEvents（地图事件处理）

**文件路径：** `src/components/map/composables/useMapEvents.js`

**引入方式：**
```javascript
import { useMapEvents } from '@/components/map/composables/useMapEvents';
```

**参数（Options）：**
```javascript
useMapEvents({
  mapTransform,      // useMapTransform 返回的对象
  mapTooltip,        // useMapTooltip 返回的对象
  mapContainerRef,   // 地图容器 DOM 引用
  emit               // 组件的 emit 函数
});
```

**返回的方法（Functions）：**

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `handleMouseDown(e)` | `MouseEvent` | `void` | 鼠标按下 → 开始拖拽 |
| `handleMouseMove(e)` | `MouseEvent` | `void` | 鼠标移动 → 更新坐标提示 + 继续拖拽 |
| `handleMouseUp()` | 无 | `void` | 鼠标释放 → 结束拖拽 |
| `handleMouseLeave()` | 无 | `void` | 鼠标离开 → 结束拖拽 + 隐藏提示 |
| `handleWheel(e)` | `WheelEvent` | `void` | 滚轮缩放 |
| `handleMapClick(e)` | `MouseEvent` | `void` | 地图点击，触发 `emit('map-click', { x, y })` |

### 2.3 useMapTooltip（悬浮坐标卡片）

**返回的方法（Functions）：**

### 2.4 usePoiHandlers（POI 事件处理）

**文件路径：** `src/components/map/composables/usePoiHandlers.js`

**引入方式：**
```javascript
import { usePoiHandlers } from '@/components/map/composables/usePoiHandlers';
```

**参数（Options）：**
```javascript
usePoiHandlers({
  poiStore,      // usePoiStore() 实例
  emit,          // 组件的 emit 函数
  mapTransform   // useMapTransform 返回的对象
});
```

**返回的方法（Functions）：**

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `centerOnPoi(poi)` | `poi: Object` | `void` | 让地图以平滑动画将 POI 居中定位到屏幕中心 |
| `handlePoiClick(poi)` | `poi: Object` | `void` | POI 点击：切换选中状态，自动居中并打开弹窗 |
| `handlePoiHover(poi)` | `poi: Object` | `void` | POI 悬停，触发 `emit('poi-hover', poi)` |
| `handlePoiLeave(poi)` | `poi: Object` | `void` | POI 离开，触发 `emit('poi-leave', poi)` |

> **`centerOnPoi(poi)` 内部逻辑：**
> 1. 计算目标偏移量：`targetX = window.innerWidth/2 - poi.x * scale`
> 2. 应用边界限制，防止地图被拖出可视区域
> 3. 开启 0.4s 的 CSS 平滑过渡动画
> 4. 动画结束后恢复即时拖拽响应

## 3. 全局工具与配置

### 3.1 request（Axios 封装）

**文件路径：** `src/utils/request.js`

**默认导出：** `request`（axios 实例）

**配置：**
- `baseURL`: `/api`
- `timeout`: 10000ms
- 自动携带 `Authorization: Bearer <token>`（从 `localStorage.token` 读取）

**主要功能：**
- 请求 / 响应日志打印
- 统一错误处理（400/401/403/404/500 等）
- 自动解包 `response.data`（返回 `response.data` 而非完整 response）

**引入方式：**
```javascript
import request from '@/utils/request';
// 使用方式同 axios
request.get('/pois');
request.post('/pois', data);
```

## 4. Provide / Inject 全局方法

### 4.1 `close`（弹窗关闭函数）

**提供方：** `src/components/Dialog/Dialog.vue`

**注入方式：**
```javascript
import { inject } from 'vue';
const close = inject('close');
close(); // 关闭当前 Dialog 弹窗
```

**说明：** 子组件通过 inject 获取 Dialog 弹窗的关闭函数，用于在表单提交成功后主动关闭弹窗。

## 附录：常见调用场景

### 场景一：在任意组件中定位到某个 POI

```javascript
import { useMapStore } from '@/stores/mapStore';
const mapStore = useMapStore();

mapStore.centerOnPoi(targetPoi);
```

### 场景二：在任意组件中获取 POI 列表

```javascript
import { usePoiStore } from '@/stores/poiStore';
const poiStore = usePoiStore();

// 触发加载（如果还没加载过）
await poiStore.fetchAllPois();

// 访问列表
console.log(poiStore.pois);
```

### 场景三：打开自定义弹窗

```javascript
import showDialog from '@/components/Dialog/Dialog';
import MyComponent from '@/components/MyComponent.vue';

showDialog(
  { title: '弹窗标题', left: 100, top: 200, draggable: true },
  MyComponent,
  { customProp: 'value' },
  'clearAllAndShow'  // 模式：关闭所有旧弹窗，再打开新的
);
```
